> 为视频创作者设计的「隐形 + 懂你节奏」的 macOS 提词器。

这是**内部小范围测试版**（非正式发布）。
- 当前使用共享测试配额的火山引擎 ASR。

感谢你愿意帮忙试用！你的反馈对这个产品非常关键。

## 核心价值（我们最想验证的 3 件事）

1. **窗口完全不被录到**  
   提词器悬浮在屏幕上，但 OBS / 录屏 / 屏幕共享 / QuickTime 都看不到它。

2. **节奏真正跟你走**  
   你先自然念一遍稿子 → AI 生成你的个人节奏 → 之后提词器严格按照你自己的语速、停顿、呼吸自动滚动 + 逐字高亮。

3. **口述模式（空稿件也能用）**  
   什么都不写，直接开始录音 → 念完后自动把你说的话变成带节奏的稿件，之后就能按这个节奏跟读。

## 快速安装与启动

1. 打开 `MyPace-Preview-0.8.0.dmg`
2. 把 **MyPace Preview** 拖到「应用程序」文件夹
3. 双击启动

**首次启动注意事项：**

- 会弹出「无法验证开发者」→ 右键 App → 「打开」→ 再次确认。

- 第一次会看到欢迎引导页（3 步介绍核心功能）。

## 推荐试用流程（建议按这个顺序）

### 第 1 轮：验证「窗口隐形」（1 分钟）
1. 启动后看到半透明浮动窗口。
2. 用 ⌘⇧5 或 OBS 录一段屏幕（包含 MyPace 窗口）。
3. 回放视频，确认 MyPace 窗口**完全没有出现**。

**请特别注意**：在不同录制软件（OBS、CapCut、系统录屏、Loom 等）下的表现。

### 第 2 轮：验证「节奏同步 + 逐字高亮」（最重要，5-8 分钟）

**有稿件的情况（念稿模式）：**
1. 点击右上角「编辑」或按 ⌘E，粘贴一段你常用的 300-800 字稿子（建议用你自己写的内容）。
2. 点击底部大按钮 **「开始练习录音」**（或按 ⇧⌘R）。
3. **正常、自然地念一遍**（带停顿、语气、思考都可以）。
4. 念完点击「结束录音」。
5. 等待 ASR 对齐（会显示进度，通常 5-12 秒）。
6. 对齐完成后，点击 **「按节奏播放」**。
7. 再念第二遍，这次**让提词器跟着你走**，感受它是否跟得上你的节奏和停顿。
8. 注意观察**逐字高亮**效果是否自然、不抖。

**空稿件的情况（口述模式，强烈建议试）：**
1. 新建一个空白脚本（什么都不写）。
2. 直接点击「开始练习录音」。
3. 随意说一段你想讲的内容（1-2 分钟即可）。
4. 结束后等待 ASR。
5. 看它是否自动把你说的话变成了带节奏的稿件，并能按这个节奏回放。

### 第 3 轮：真实使用场景模拟（可选但很有价值）
- 用你正在准备的真实视频脚本，走完整「练习一次 → 正式录制时用节奏模式」的流程。
- 尝试在录制视频的同时使用提词器（模拟真实工作流）。

## 当前操作界面（v0.8）

浮动窗口底部有明显按钮，不用死记快捷键：

- **上下箭头**：上一句 / 下一句
- **中间大按钮**：根据状态自动变化（开始录音 / 结束录音 / 按节奏播放 / 暂停）
- 右上角工具栏：编辑稿件、切换稿件、设置

常用快捷键（hover 按钮也会提示）：
- `Space`：主要动作触发（录音/播放/暂停）
- `↑ / ↓`：上下句
- `⇧⌘R`：开始练习录音
- `⌘E`：编辑当前稿件
- `⌘,`：打开设置

设置里可以调整：字号、透明度、强调色、阅读引导线、是否允许被录屏（调试用）。

## 已知限制（请提前知道）

- **没有节奏可视化编辑器**：ASR 对齐不准时，目前只能重新练习录音，无法手动拖时间戳微调。
- **ASR 使用共享测试配额**：本次内测大家共用一份配额，用完后练习功能会暂时失效。我会尽量监控并在用完前通知大家。
- **没有完整的视觉自定义**：字号、透明度、颜色可以调，但还比较基础。
- **长稿件 / 复杂停顿** 时的对齐效果还未充分优化。
- 目前仅支持 Apple Silicon Mac + macOS 14+。

## 隐私与数据

- 练习录音的音频会上传到火山引擎做对齐（过程有进度提示）。
- 对齐完成后，原始音频**只保存在你本机** `~/Library/Application Support/MyPacePreview/`。
- 本 app 除 ASR 外无其他网络请求，无埋点，无崩溃上报。
- 你可以随时 `rm -rf ~/Library/Application\ Support/MyPacePreview/` 完全清除数据。
