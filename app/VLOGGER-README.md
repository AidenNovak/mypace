# MyPace Preview v0.4 · 给 vlogger 的早期试用版

> 一台不会被录到，还能"听懂你节奏"的 macOS 提词器。

---

## 这是什么

**MyPace** 专门为 vlogger 设计，解决两件事：

1. **窗口不被录到** —— 提词器悬浮显示稿件，但 QuickTime / OBS / Loom / Zoom 屏幕共享 **任何屏幕录制都看不到它**
2. **节奏跟你走，不是你跟节奏走** —— 你先自然念一遍，AI 生成节奏，之后照着你的节奏自动滚动

这是 **v0.2 Preview 版**，让你**完整体验核心价值**，再决定要不要等正式版。

---

## 5 分钟试用流程

### 1. 安装（30 秒）

1. 双击 `MyPace-Preview-0.2.0.dmg`
2. 把 **MyPace Preview** 拖到 **Applications**
3. 在 Launchpad 找到 **MyPace Preview**，双击启动

⚠️ **首次启动弹"无法验证开发者"提示**：
- 右键点击 app → 选「打开」→ 弹窗里再次确认「打开」
- 这是因为没上 App Store + 没做苹果公证（要付 ¥688/年）

⚠️ **首次启动会弹麦克风权限请求** —— 录音用，可以拒绝（拒绝后只能用静态提词器）。

### 2. 体验「窗口不被录到」（30 秒）

1. 启动后看到一个半透深色浮动窗口
2. 按 <kbd>⌘ Shift 5</kbd> → 录制屏幕 → 录几秒 → 停止
3. 回放视频：**MyPace 窗口完全不在画面里** ✨

### 3. 体验「节奏映射」（最有价值，3 分钟）

这是 MyPace 真正的差异化：

1. <kbd>⌘E</kbd> 把你自己的稿子粘进来（每行一句）
2. <kbd>⇧ ⌘ R</kbd> 开始录音 —— **正常自然地念一遍稿子**
3. 念完按 <kbd>Space</kbd> 结束录音
4. 等 5-10 秒（火山引擎对齐中）
5. ✨ **完成后自动开始按你的节奏自动滚动** —— 提词器跟着你刚才的语速走

念第二遍的时候，提词器会按你第一遍的节奏滚动 —— 你不用再"追"提词器，是提词器在配合你。

## 操作方式

**v0.4 起所有操作都有可见按钮**，不用记任何快捷键。

### 浮窗底部的按钮

```
[ ↑ 上一句 ]  [ ↓ 下一句 ]      [   ⏺ 开始录音   ]      [ ✏ 编辑 ] [ 📁 切换稿件 ] [ ⚙ 设置 ]
```

中间**大红色按钮**按状态动态变化：
- 没有节奏映射时：`● 开始录音`
- 录音中：`■ 结束录音`
- 对齐中：`正在对齐 62%`
- 有节奏映射时：`▶ 按节奏播放`
- 播放中：`⏸ 暂停`

### 喜欢键盘的人

每个按钮 hover 时都显示快捷键。常用的：

| 按键 | 作用 |
|---|---|
| <kbd>Space</kbd> | 主按钮触发（录音/播放/暂停） |
| <kbd>↑</kbd> / <kbd>↓</kbd> | 上一句 / 下一句 |
| <kbd>⌘E</kbd> | 编辑稿件 |
| <kbd>⌘,</kbd> | 偏好设置 |
| <kbd>⌘=</kbd> / <kbd>⌘-</kbd> | 字号 +/- |
| <kbd>Esc</kbd> | 取消当前动作 |
| <kbd>⌘Q</kbd> | 退出 |

## 隐私保证

- ✅ **音频处理**：录音会上传到火山引擎做对齐（每次都会显示进度条），对齐完成后**音频文件留在你本机的应用支持目录里，不上云**
- ✅ **不联网（除了 ASR）**：本 app 除了 ASR 调用之外没有任何网络请求
- ✅ **不埋点**：没有 analytics、没有崩溃上报
- ✅ **本地存储**：稿件 JSON 存在 `~/Library/Application Support/MyPacePreview/`
- ✅ **不残留**：完全可以一句 `rm -rf ~/Library/Application\ Support/MyPacePreview/` 清干净
