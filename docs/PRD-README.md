# MyPace

> **一次练习，与自己的节奏重逢**  
> 专业屏幕录制提词器 · 精确复刻个人语速、停顿与呼吸 · 零可见脚本 · 开源

MyPace 是一款为内容创作者、在线讲师、产品经理设计的**录音节奏同步提词器**。  
你只需输入文稿 → 练习念一遍 → 系统自动生成你的个人节奏映射，后续正式录制时提词器会**严格按照你自己的说话节奏**平滑滚动，让你一次过、专业、自然。

---

## 核心特性

- **个人节奏复制器**：练习录音一次，即可精确复刻你的语速、停顿、强调（支持连续两行自然呈现）
- **云端优先 + 本地兜底**：练习阶段可选高精度云端 ASR（OpenAI / 火山引擎 / 讯飞等），正式录制 100% 本地
- **零可见脚本**：提词器内容绝不进入最终录制画面（最高优先级）
- **极致简洁**：录制中 3 秒无操作自动隐藏所有控件，专注表达
- **完整可控**：可视化节奏编辑器，支持 10ms 级手动微调
- **macOS First**：v1 版本优先支持 macOS（13 Ventura 及以上，Apple Silicon 优先），Windows 支持计划在 v2
- **隐私优先**：练习音频默认对齐后自动删除，所有节奏数据本地存储

---

## 为什么选择 MyPace？

市面大多数提词器只提供「固定速度滑块」，无法匹配真实人类的节奏变化。  
MyPace 的核心创新是：**把你已经说过的自然节奏，精确复制到下一次**。

告别重录、眼神飘、卡顿，让每一次正式录制都像和自己对话一样流畅。

## 技术栈（规划）

- 桌面框架：Tauri 2.0（Rust + WebView）或 Electron + Rust 子进程
- 音频处理：系统原生录制 + whisper.cpp（本地兜底）
- 云端对齐（可选）：OpenAI Whisper API / 火山引擎 / 讯飞（练习阶段）
- 文本对齐算法：Levenshtein / Needleman-Wunsch + 置信度评估
- 窗口捕获排除：macOS ScreenCaptureKit（v1 专精实现，零捕获）
